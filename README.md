# Equiptrack (fullstack)

This folder contains both the backend and frontend in one project:

- `backend/` - Node.js + TypeScript + Express API (port **4000**)
- `frontend/` - Vite + React + TypeScript UI (port **5173**)

## Quick start

From this root folder:

```bash
npm run install:all
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:4000/api/products
